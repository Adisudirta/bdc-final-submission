
# Trash Cash Web App

Sell your trash, get your cash

## Installation

### 1. Install all dependencies
```terminal
npm install
```

### 2. Prisma generate
```terminal
npx prisma generate
```

### 3. Prisma migrate
```terminal
npx prisma migrate dev
```

### 4. Run seeder
```terminal
npm run seed
```

### 5. Run the project!
```terminal
npm run dev
```
